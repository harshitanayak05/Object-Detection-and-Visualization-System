# Object-Detection-and-Visualization-System
A real-time object detection and visualization system using Arduino, Java for radar like graphical visualization.
This system detects nearby objects, measures their distance, and visually represents their position using a radar-like graphical interface. The servo motor continuously rotates from 0° to 180° for environmental scanning, while the ultrasonic sensor measures object distance in real time.

The measured angle and distance values are transmitted from Arduino to Java through serial communication for live visualization and monitoring.

## Features
- Real-time object detection
- Distance measurement using ultrasonic sensor
- 180° environmental scanning
- Radar-like graphical visualization
- Serial communication between Arduino and Java
- Low-cost and efficient implementation

## Hardware Used
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- Servo Motor
- Breadboard
- Jumper Wires

## Software Used
- Arduino IDE
- Java (Eclipse IDE)
- Serial Communication Library

## Working Principle
1. Servo motor rotates from 0° to 180°
2. Ultrasonic sensor detects nearby objects
3. Distance and angle data are collected
4. Arduino sends data through serial communication
5. Java processes and visualizes the data
6. Radar interface displays object position in real time
